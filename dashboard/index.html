            About the data

            What one row means

            A row is one recorded sighting event from the underlying feed, not a census of all whales in the area. Some rows report a single whale, while others report a group observed together.

            What counts mean

            no_sighted reflects how many animals were reported in that observation when a usable count was present. More rows or more points on the map should not be read as a direct measure of whale abundance.

            What gets filtered out

            The dashboard only maps positive sightings with usable coordinates. Blank counts, invalid dates, impossible coordinates, and unresolved source noise are handled in the cleaning pipeline before the public view is built.

            Why some whales stay uncertain

            Some source labels are too broad to safely present as a confirmed whale species. Those records stay unresolved or caution-marked so the dashboard does not imply more certainty than the data supports.

            How sources should be read

            Source names combine witness channels, apps, and feeds. They help explain where observations came from, but different sources may follow different reporting patterns and should not be treated as equal sampling effort.

            How to use this responsibly

            Use the dashboard to explore patterns, seasonality, and distribution. Treat it as an interpreted sightings atlas, not a population estimate or a complete record of every whale in each region.
